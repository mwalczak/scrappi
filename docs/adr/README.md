# Architecture Decision Records (ADRs)

This directory contains Architecture Decision Records for the Scrappi project.

## What are ADRs?

ADRs document important architectural decisions made in the project, including:
- Context: What prompted the decision
- Decision: What was decided
- Consequences: Implications of the decision

## Active ADRs

- [ADR-001: Hexagonal Architecture with DDD](./001-hexagonal-architecture-with-ddd.md) - Foundation architecture using Hexagonal Architecture (Ports & Adapters) with Domain-Driven Design principles

## Status Legend

- **Accepted**: Currently in use
- **Proposed**: Under consideration
- **Deprecated**: No longer applicable
- **Superseded**: Replaced by another ADR
